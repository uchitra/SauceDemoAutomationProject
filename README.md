# Sauce Demo Automation Project
This Project is a Selenium automation framework developed using Java.
It automates the Sauce Demo website to test login, add to cart,checkout flow, and negative login scenarios.

## Technologies Used
Java
Selenium WebDriver
Maven
Git and GitHub
Eclipse

## Automated Test Scenarios
Valid Login Test
Add Products to Cart
Cart Verification
Checkout Process
Negative Login Test(Invalid Username and Password)

## Project Structure
LoginAndCart.java
NegativeLoginTest.java
pom.xml

## How to Run the Project
Clone the repository from GitHub
open the project in Eclipse IDE
Configure ChromeDriver
Run the test classes
